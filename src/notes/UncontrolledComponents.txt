01) 'Na maioria dos casos, recomendamos o uso de componentes controlados para implementar formulários. Em um componente controlado,
    os dados de formulário são manipulados por um componente React. A alternativa são componentes não controlados,
    onde os dados de formulário são controlados pelo próprio DOM. 
    
    Para criar um componente não controlado, em vez de criar um manipulador de evento para cada atualização de estado,
    você pode usar ref para obter os valores do formulário do DOM.
